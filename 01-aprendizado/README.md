# 🧠 01 — Aprendizado

Skills para aprender e dominar o Claude Code no seu ritmo.

---

## Skills disponíveis

### [/learn](./learn/)

A skill de onboarding completa. Diagnostica seu nível, oferece exercícios práticos, quiz e roadmap personalizado.

**Como usar:**

```bash
/learn              # diagnóstico completo
/learn quiz         # modo quiz (5 perguntas)
/learn path         # ver roadmap
/learn hooks        # exercício: hooks
/learn agents       # exercício: agentes
/learn claude-md    # exercício: CLAUDE.md
/learn skills       # exercício: criar skills
/learn commits      # exercício: commits
```

**Arquivos:**

| Arquivo | Função |
|---------|--------|
| `learn.md` | Roteador principal — interpreta o comando e distribui |
| `learn-diagnose.md` | Inspeciona o ambiente e detecta seu nível |
| `learn-exercise.md` | Oferece exercícios práticos por nível e tópico |
| `learn-quiz.md` | Quiz com 15 perguntas em 3 níveis |
| `learn-path.md` | Roadmap visual com próximos passos |
